Glossary

Click on a term to reveal its definition

  • API
    • An Application Programming Interface is a set of protocols for accessing features of one software application from another.
  • Access Token
    • An access token is a security key that allows the holder to access a program's API. You should not issue tokens to organizations you do not trust! To see tokens you have issued, log into Canvas, and visit your account settings. All tokens created by this app will be named Single_Use_Rubric_Export. You can delete a token manually by clicking on the trash can icon to the right of its name.
  • Single-use token
    • The tokens you authorize for this application are temporary in the sense that they are not recorded outside of Canvas. Once you delete them, as recommended, they will not be stored anywhere.
  • Reusable token
    • Some applications, such a Peer Evaluation and iClicker, store authorization tokens in a database for re-use. Such applications require only a single authorization action, which may even have been taken for you by a Canvas administrator.
  • Instructor View
    • When an instructor visits an course in which this tool has been installed, he will see a new link in the course menu. Students will never see that link when logged in as themselves.
  • External Tool (aka App)
    • An application hosted outside of Canvas that conforms to the IMS LTI standard for interoperability. External Tools are installed by visiting the Apps tab within course Settings.
  • Logging out
    • Logging out of an LTI tool destroys the session data associated with that tool. In the case of the Rubric Export tool, it also deletes the single-use token.

Overview

When you choose to use a rubric in the context of a Canvas assignment, there is no easy way to perform statistical analyses of student performance on individual components. Currently the Canvas gradebook export data only includes the total grade for each assignment. This tool was built as a (hopefully) temporary work-around for those who need want more detailed information.

Each time you enter an assignment url and click export, you will initiate a download of a .csv data that includes the most recently submitted rubric grades for each student. If you need data from more than one assignment, you will have to perform multiple exports. Aggregating data is left to the user.

Step-by-Step

  • Installation
    1. Visit https://www.udel.edu/sakai/lti/secrets.html and locate the key : secret for the Rubric Export tool. You will need to provide your UDelNetID and password to access the page.
    2. Launch Canvas, and browse to any course for which you have instructor access.
    3. Click the Settings link located at the bottom of the Canvas navigation menu, located at the left side of the screen.
    4. Click on the Apps tab.
    5. Click the Export Rubric Scores button.
    6. Click the blue Add Tool button.
    7. In the window that pops up, enter the key and secret.
    8. Refresh your browser, and look in your Canvas course menu for a new link. Only instructors and administrators can see the Export link, although it does not appear gray.
  • Export data
    1. In the Canvas navigation menu, click the Export Rubric Scores link. A blue U of Delaware Canvas access request will appear. Click log in to authorize access to your Canvas data.
      1. This action will generate a Canvas API (Application Program Interface) authorization token. This is a single-use token, which will not be stored, and which you should delete (see below) once you have finished downloading your data. Because the token is not stored, you will need to re-authorize after your app session has ended. This will occur when you:
        1. Delete the token
        2. Close your browser
        3. Log out of any other UD hosted LTI app, such as Peer Evaluation, Post'Em, or reThinQ.
    2. Once you have authorized, you will see a very simple web form. Paste the url of any Canvas Assignment that includes a scoring rubric into the Assignment URL field and click 'extract'.
    3. Course ID and Assignment ID are extracted from the URL. These should appear as 5 - 10 digit numbers. If there is anything other than numbers in the fields, check your assignment url and try again.
    4. If your ID numbers look okay, click submit to initiate the download.
    5. Repeat as necessary to obtain data from other assignments. Data need NOT come from the particular course in which you installed the app.
    6. When finished, use the 'Delete token' button to delete the token. Failure to do so will result in a clutter of useless tokens in your personal settings, but unless you leave your computer unattended, will not create a security issue.

FAQ

Click on a question to reveal its answer

  • How can I be sure that students will never be able to obtain data using this tool?
    • There are multiple safeguards for this. First, students will not be able to see the link. Second, students who click the link and authorize data access will not be able to access any data visible only in instructor view. For example, a student token does not have the power to reveal other students' grades.
  • Do I have to re-install every semester?
    • No. As long as you have access to the course in which you installed the app, you will be able to use it. At the end of the semester, it is a good idea to unpublish your course, but that should not prevent you from exporting data.
  • Why does this tool appear in a course menu, rather than in my personal menu?
    • At some point, we may decide to install this for everyone, at which time we would move it to the User menu rather than the course menu. As of this date, we have chosen to make the tool available only for those who choose to install it. Canvas does not have a mechanism for installing User menu items individually.
