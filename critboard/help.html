Glossary

Click on a term to reveal its definition

  • Rubric Tabs
    • Rubric tabs divide the critboard page into any number of sub-divisions. Each tab is used to critique images along the dimension indicated by the tab's label. Since Critboard rubrics are intended to emerge during an in-class discussion, there is no relationship between the Critboard rubric and any Canvas rubric you may associate with the assignment, nor will your sort scores be entered as Canvas rubric scores.
  • Save Crit
    • Use this button to preserve a copy of your sorts. Clicking this button will place an html page containing your current tabs in the Canvas File folder for your course. This button is available only in Instructor View.
  • Compile Grades
    • Use this button to push grades into the Canvas gradebook, based on the current sort status of the images. 5 points are awarded for each top shelf appearance, 3 for middle, 1 for bottom. This button is available only in Instructor View.
  • Full View
    • When you mouse over a thumbnail image, a full view link appears below it. Click this link for a better view of the image. The full view window is resizeable and draggable, and includes radio buttons for re-shelving, as well as a delete button. Deleting an image in one tab will automatically remove ALL copies of that image from all tabs.
  • Student View
    • When a student enters the Critboard tool, he does NOT have an opportunity to perform a critique. He/she will see a grade distribution for the assignment if it has been graded, and links to any sorts that have been saved using the Save Crit feature.
  • Instructor View
    • Only instructors can perform sorts, however, it is highly recommended that you do this during class, so that students can benefit from your reasoning, suggest new dimensions for critique, etc.
  • Feature Requests
    • Are encouraged. You must be logged into our Google site to make a request.

Overview

Critboard for Canvas@UD is designed to be used in conjunction with a Canvas Assignment requiring students to upload an image as their assignment submission. Submitted images can then be sorted based on criteria established during class. Sorting images during class gives all students a chance to better understand the criteria along which their submissions will be judged. The Critboard tool also streamlines the grading process, by assigning points for each category based on the shelf it is placed on.

How to Use this Page

The glossary on the left should help you understand what the features of the tool are, and how these are intended to be used. Step-by-Step instructions for setting up and using the Critboard as well as an FAQ can be located in the accordion panels below. Click on a panel's title to expand it.

Step-by-Step

  • Installing Critboard (at UD)
    1. Launch Canvas, and browse to the course in which you wish to use the Critboard
    2. Click the Settings link located at the bottom of the Canvas navigation menu, located at the left side of the screen.
    3. Click on the Apps tab.
    4. Click the Critboard button, and take careful note of the key and secret, located at the bottom of the App description text.
    5. Click the blue Add Tool button.
    6. In the window that pops up, enter the key and secret.
    7. Refresh your browser window, and check to see that a Critboard link has been added to the Canvas navigation menu.
      1. To re-order the links in the navigation menu, visit the Navigation tab and drag the links to the desired locations.
  • Installing Critboard (at xxx.instructure.com)
    1. Request a key : secret by completing this form. You will receive a key and secret via email. Please allow a few days for processing, as requests are vetted manually. UD LTI Apps are not available to for-profit institutions.
    2. Once you have a key and secret, copy this xml configuration url onto your clipboard: https://apps.ats.udel.edu/canvas/critboard/critboard_shared.xml
    3. Follow the instructions here: How do I configure an External App for a course using a URL?
      1. You will be asked to authorize use of a course-specific API token. You must accept the authorization request to use the app. Only click the accept (login, OK, submit...) button ONCE. There may be a brief delay while the process is completed.
      2. When the installation is complete, you will see a mostly blank page asking you to select or create an assignment requiring a file upload submission. You are now ready to proceed to the next step.
  • Create an Assignment
    1. In the Canvas navigation menu, click the Assignments link. This should open even if the link appears gray.
    2. Click the blue +Assignment button.
    3. In the Create Assignments UI locate the Submission Type selector, and choose Online. Under entry options, select File Uploads.
      1. Optionally restrict File upload types to permite jpg, jpeg, gif, png.
    4. Configure the remaining assignment options as desired.
    5. When you are ready, publish the assignment. Unpublished assignments will not be visible to students, even though as the instructor, you can see them.
  • Critique submitted images
    1. Click on the Critboard link in the navigation menu.
    2. Select an assignment from the assignnment drop-down list. Only assignments that accept file upload submissions will appear in the list.
      1. Submitted images will appear as thumbnails.
    3. Enter a comma delimited list of Rubric Tab titles in the Rubric Elements field. For instance, if you wish to critique submissions based on lighting, composition and use of negative space, your tab titles might be lighting, composition, negative space.
    4. Click the Create Rubric Tabs button.
      1. Tabs will appear. Click tabs to confirm that a copy of each uploaded image appears in each tab.
    5. Sort the images in each tab based on the criteria you have chosen. Best images should go on the top shelf, weaker ones on the bottom shelf. You can drag the images from shelf to shelf with your mouse, or use the open full link (visible on mouse-over) to view a larger version of any single image, and choose a shelf radio button.
    6. Additional tabs can be added at any time by entering a single tab label, and clicking Add Tab.
  • Save your work

    Once you have finished your critique, there are several ways to preserve a record of what you have done.

    1. The Save Crit button will save a copy of your tabs to the Canvas file system. Links to each saved sort will be generated in the Student View of the Critboard Tool.
    2. The Download Crit button can be used to download a copy of your tabs in an html file.
    3. The Compile Grades tab will award each student 5 points for each top shelf appearance of his/her submitted image, 3 for the middle shelf, and 1 for the bottom. Grades will be posted directly to the Canvas gradebook. Rubric scores will appear as comments in the gradebook and speed grader.
      1. If you do not see a Compile Grades button, it is likely that one or more of your students submitted multiple images. You must delete the extra image or images before compiling grades. To delete an image, locate the delete button, visible only in 'full view'.

FAQ

Click on a question to reveal its answer

  • I created an assignment with a submission type of file upload, and I can see it when I launch the Critboard, but my students can't see it. What's wrong?
    • Be sure your assignment is published.
  • Why doesn't the Critboard create rubric tabs to match the rubric I created for my assignment in Canvas?
    • This tool was designed for faculty who wish to critique images interactively with students. The philosophy is that rubrics should not be carved in stone before the critique even begins. We recommend NOT providing a Canvas rubric at all when creating an assignment to be used with this tool.
